# DL Journey Tracker 🚗

## LL to Permanent License System — OOP Mini Project

---

## 📌 Overview
DL Journey Tracker is a full-stack application that helps users track their journey from Learner License (LL) to Permanent Driving License (DL).

---

## 🎯 Features
- Stage Tracking (5 stages)
- Days Remaining Calculator
- Document Checklist (DB-based)
- Alert System (Green/Yellow/Red/Expired)
- RTO Test Guide
- C++ OOP Integration

---

## 🧠 OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Classes & Objects

---

## ⚙️ Tech Stack
- C++ (Core Logic)
- Node.js + Express (Backend)
- MySQL (Database)
- HTML, CSS, JavaScript (Frontend)

---

## 🔄 System Flow
Frontend → Node.js → C++ → JSON → Frontend

---
