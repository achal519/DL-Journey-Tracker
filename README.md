# DL Journey Tracker 🚗

## LL to Permanent License System — OOP Mini Project

### Problem Statement
Every year lakhs of Indians get their Learner License but never 
convert it to a Permanent DL because the process is confusing 
and no system guides them through it.

### What This System Does
- Tracks exact stage in LL to DL journey (5 stages)
- Calculates days remaining before LL expires
- Generates personalized document checklist
- Gives color-coded alerts (Green/Yellow/Red/Expired)
- Provides RTO test preparation guide

### Why It Is Original
mParivahan shows DL info but provides:
- No stage tracking
- No deadline alerts
- No personalized checklist
- No test preparation guide

### OOP Concepts Used
- Classes and Objects
- Encapsulation
- Inheritance (License → LearnerLicense, PermanentLicense)
- Polymorphism (displayDetails override)
- Abstraction
- Constructors
- Virtual Functions
